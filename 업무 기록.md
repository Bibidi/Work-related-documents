## DNS
### DNS 레코드 수정
- 메뉴얼을 참고해도 되고, Jira - APAC GDC 기록을 보고 따라해도 된다.
- 끝나면 DNS 폴더에 작업 기록 남기고 또, 북마크의 업무기록에 기존 양식을 따라서 기록 남기면 된다(공통).

### 신규 DNS 추가
- 가장 먼저 참고해야 될 것은 Samba -> 4.LOTTE -> OPS-GDC-NOC -> 나건우 -> dns 업무에서 가장 최근에 작업한 내역을 확인하는 것이다. 이 아이피 주소를 기준으로해서 //?!?@ 이런 식으로 되어 있는 파트 위쪽에 신규 DNS를 추가한다.
- 수정 1차의 경우 씨디네트웍스 구글 시트 기준으로  첫 번째 아이디로 접속해서 작업하는 창. 수정 2 ~ 4는 새 cmd 창을 띄우고 구글 시트 아이디 순서
- 끝나면 DNS 폴더에 작업 기록 남기고 또, 북마크의 업무기록에 기존 양식을 따라서 기록 남기면 된다(공통).

#### 공통
업무 기록 마지막에 contents에 어떤 링크를 채워야 하는데, 이건 outlook 메일에서 첫 작업 요청 메일 상단에 보면 티켓 번호가 있을 것이다. 여기에 접속해서 나오는 링크를 기입하면 된다. 이 링크로 들어가서 올바른 화면을 보려면 SecureLink를 켜야 한다.

- 오후 6시 퇴근 이후 본사로 날아가는 통화가 우리 쪽으로 돌아오도록 바뀐다(바뀌면서 알람도 울림). 그때부터 전화 통화로 긴급한 업무가 발생하면 에이비클 단톡방으로 상황 알릴 것
