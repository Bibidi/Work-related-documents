# 2. 파일 업로드
- 수정할 파일을 왼쪽 상단에 위치한 파일 업로드 버튼을 눌러 웹에 업로드 합니다.

![[Pasted image 20260130112417.png]]
- 파일 업로드 버튼을 눌렀을 때 나오는 파일 탐색기입니다. 이 창을 통해 수정하고자 하는 파일을 찾아서 선택한 후 열기 버튼을 누릅니다.


![[Pasted image 20260130112600.png]]
- 파일이 열리면 기존 엑셀 프로그램에서 사용하는 것과 같은 화면이 나옵니다.


# 편집
![[Pasted image 20260130112828.png]]
- 파일 시스템에서 한 파일의 권한은 보통 읽기, 쓰기, 실행 세 가지로 나눠집니다. 윈도우에서는 대부분의 파일에 대해 사용자가 세 가지 권한을 기본적으로 가집니다. 그래서 파일을 대부분 자유롭게 읽고 쓰고 실행할 수 있습니다.
- 그러나 웹 엑셀은 파일을 업로드 한 뒤 열면 기본적으로 읽기 권한만 가진 채 열립니다.
- 파일을 수정하려고 시도해 봅니다. 그러면 수정이 되지 않고 상단에 '**읽기 전용** 이 프로젝트는 읽기 전용 모드로 열렸습니다'라는 알람이 발생합니다.


![[Pasted image 20260130114847.png]]
- 읽기 전용 모드 알람 옆에 위치한 통합 문서 편집을 눌렀을 때 나오는 화면입니다.
- 변환을 눌러 최신 파일 형식으로 변환합니다.

![[Pasted image 20260130115150.png]]
- 이제 자유롭게 편집이 가능합니다.


# 기존 편집한 파일 찾기 및 내보내기

## 파일 히스토리
![[Pasted image 20260130115601.png]]
- 웹 엑셀 첫 화면에서 밑으로 조금 내려오시면 **내 통합 문서**를 찾을 수 있습니다. 여기서 기존에 업로드한 파일을 확인하실 수 있습니다.

## 파일 내보내기
![[Pasted image 20260130115917.png]]
- 왼쪽 상단에서 파일 - 내보내기를 선택하시면 수정한 파일을 다운로드 할 수 있습니다. 원하는 옵션을 선택하신 뒤 다운로드 하시면 됩니다.

![[Pasted image 20260130120129.png]]
- 다운로드를 하시면 다운로드 폴더에 위와 같이 파일이 다운로드 됩니다.
