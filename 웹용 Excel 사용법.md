# 2. 파일 업로드
- 수정할 파일을 웹용 Excel에 업로드합니다. 화면 왼쪽 상단의 파일 업로드 버튼을 누릅니다.

![](./Attached%20files/Pasted%20image%2020260130112417.png)
- 파일 탐색기 창이 열리면, 수정하려는 파일을 선택한 뒤 열기 버튼을 누릅니다.


![](./Attached%20files/Pasted%20image%2020260130112600.png)
- 파일이 열리면, 기존 엑셀 프로그램과 유사한 편집 화면이 표시됩니다.


# 읽기 전용으로 열릴 때 편집 가능하게 전환
![](./Attached%20files/Pasted%20image%2020260130112828.png)
- 파일 시스템에서 파일 권한은 보통 읽기, 쓰기, 실행으로 구분됩니다. 윈도우 환경에서는 대부분의 파일을 사용자가 자유롭게 열고 수정할 수 있기 때문에, 보통은 별다른 제한을 의식하지 않습니다.
- 하지만 웹용 Excel에서는 업로드한 파일이 상황에 따라 **읽기 전용 모드**로 열릴 수 있습니다. 이 상태에서는 셀을 수정하려고 해도 편집이 적용되지 않고, 상단에 다음과 같은 알림이 표시됩니다. ‘읽기 전용: 이 프로젝트는 읽기 전용 모드로 열렸습니다’


![](./Attached%20files/Pasted%20image%2020260130114847.png)
- 알림 옆의 통합 문서 편집을 누르면 아래와 같은 화면이 나타납니다. 여기서 변환을 선택해 최신 파일 형식으로 변환합니다.

![](./Attached%20files/Pasted%20image%2020260130115150.png)
- 변환이 완료되면 파일을 정상적으로 편집할 수 있습니다.


# 기존 편집한 파일 찾기 및 내보내기
## 기존 편집한 파일 찾기
![](./Attached%20files/Pasted%20image%2020260130115601.png)
- 웹용 Excel 첫 화면에서 아래로 조금 내려가면 내 통합 문서 섹션이 보입니다. 여기에서 이전에 업로드했던 파일을 확인할 수 있습니다.

## 파일 내보내기
![](./Attached%20files/Pasted%20image%2020260130115917.png)
- 편집을 마친 파일을 다운로드하려면 왼쪽 상단에서 파일 > 내보내기를 선택합니다. 원하는 내보내기 옵션을 선택한 뒤 다운로드를 진행합니다.

![](./Attached%20files/Pasted%20image%2020260130120129.png)
- 다운로드가 완료되면, 윈도우의 다운로드 폴더에서 파일을 확인할 수 있습니다.
